## 라즈베리파이 구조

센서
- 마그네틱 도어 센서 스위치 (MC-38)
    - 케이블 : 무극성
    - GND, GPIO에 케이블을 각각 연결하고, 풀업 저항으로 세팅하여 사용하면 0일 때 닺혀있고 1일 때 열리도록 값이 세팅
    - 사용 Pin : 4, 17, 27
    - 주의사항
        - 자석으로 되어있어 센서 간 간격이 너무 가까우면 간섭이 일어나 제대로 된 측정이 불가할 수 있음
